# いいから寝ろ.com

**説得ではなく遮断。**  
どんな相談にも論理的に、しかし投げやりに応答し、必ず「いいから寝ろ。」で締めるAI。

---

## 🌙 特徴

- **完全クライアントサイドAI**: APIキー不要、ブラウザだけで動く超軽量生成AI
- **フルスクラッチ生成**: ルールベース + パターンマッチ + ランダム組み合わせ
- **超軽量**: Vanilla TS + 最小CSS、初期ロード < 40KB
- **夜間モード**: JST 23:00-05:00で短文化・減光
- **危機語検知**: 自動で安全テンプレ + 公的窓口リンク
- **完全オフライン対応**: ネット不要、全機能がブラウザで完結
- **共有機能**: X/LINEにワンタップ投稿

---

## 🚀 デプロイ手順（Netlify）

### 1. ビルド & デプロイ

環境変数の設定は不要です（完全クライアントサイドのため）。

```bash
npm install
npm run build
```

Netlifyに接続後、自動デプロイが走ります。

### 2. 独自ドメイン

Netlify DNS設定で `いいから寝ろ.com` または `iikarasnero.com` をCNAME設定。

---

## 🛠️ ローカル開発

```bash
npm install
npm run dev
```

完全クライアントサイドなので、通常のVite開発サーバーだけで全機能が動作します。

---

## 🧠 生成AIの仕組み

完全クライアントサイドで動作する超軽量生成AI:

1. **主題抽出**: 入力から「仕事」「恋愛」「将来」などのトピックを正規表現で判定
2. **感情分析**: ネガティブ/不安/疲労などの感情をパターンマッチ
3. **論拠生成**: 主題別のテンプレートからランダム選択（夜間は短縮）
4. **構造化**: 鏡映 → 論拠 → 「いいから寝ろ。」の3段構成
5. **危機語検知**: 自傷/暴力ワードで安全テンプレート優先

APIキー不要、完全オフライン動作、超高速。
