# いいから寝ろ.com

**説得ではなく遮断。**  
ブラウザで動く本物のニューラルネットAIが、論理的に、しかし投げやりに応答し、必ず「いいから寝ろ。」で締める。

---

## 🌙 特徴

- **真のブラウザベースニューラルネットAI**: Transformers.js + Qwen2.5-0.5B-Instruct
- **完全プライバシー保護**: すべての処理がブラウザ内で完結(サーバに送信なし)
- **APIキー不要**: OpenAI等の外部APIを一切使用しない
- **GPU不要**: CPUのみで動作(量子化モデル使用)
- **初回ロード**: 15-30秒(約150MB、その後キャッシュ)
- **推論速度**: 1-3秒/応答
- **夜間モード**: JST 23:00-05:00で短文化・減光
- **危機語検知**: 自動で安全テンプレ + 公的窓口リンク
- **PWAオフライン対応**: モデルキャッシュ後は完全オフライン動作
- **共有機能**: X/LINEにワンタップ投稿

---

## 🚀 デプロイ手順(Netlify)

### 1. デプロイ(環境変数不要)

```bash
npm install
npm run build
```

Netlifyに接続後、自動デプロイが走ります。

**環境変数は一切不要です**(すべての処理がクライアントサイド)

### 2. 独自ドメイン

Netlify DNS設定で `いいから寝ろ.com` または `iikarasnero.com` をCNAME設定。

---

## 🛠️ ローカル開発

```bash
npm install
npm run dev
```

**即座に動作します**(APIキー・環境変数不要)

初回アクセス時にAIモデル(約150MB)がダウンロードされます。

## 🧠 技術スタック

### AIエンジン
- **Transformers.js**: HuggingFaceモデルをブラウザで実行
- **モデル**: Qwen2.5-0.5B-Instruct (量子化版)
- **推論**: WebAssembly + ONNX Runtime
- **キャッシュ**: IndexedDB経由でローカル保存

### フロントエンド
- **Vanilla TypeScript**: フレームワークレス
- **Vite**: 超高速ビルドツール
- **PWA**: Service Worker + Manifest

### デプロイ
- **Netlify**: 静的ホスティング + Edge Functions(未使用)
- **CDN**: 世界中で高速配信
